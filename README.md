# MiniLisp Interpreter
This is my final project of Compiler  

## Installation

1. Navigate to the directory containing the `final` folder:  
```bash
cd [DownloadFolder]
```  
2. Install the required packages:  
```bash
pip install ply
pip install AST
```

## Implement

Run `main.py`  
```bash
python3 ./final/main.py your/path/to/test/data/*.lsp
```
